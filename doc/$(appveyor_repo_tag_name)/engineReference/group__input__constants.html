enum InputActionType

Device Event Action Types.

Enumerator
SI_MAKE 

Button was depressed.

SI_BREAK 

Button was released.

SI_MOVE 

An axis moved.

SI_REPEAT 

A key repeat occurred. Happens in between a SI_MAKE and SI_BREAK.

SI_VALUE 

A value of some type. Matched with SI_FLOAT or SI_INT.
