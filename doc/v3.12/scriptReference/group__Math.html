virtual float Global::getRandom ( (int a=S32_MAX, int b=S32_MAX)  )
virtual

Returns a random number based on parameters passed in..

If no parameters are passed in, getRandom() will return a float between 0.0 and 1.0. If one parameter is passed an integer between 0 and the passed in value will be returned. Two parameters will return an integer between the specified numbers.

Parameters
aIf this is the only parameter, a number between 0 and a is returned. Elsewise represents the lower bound.
bUpper bound on the random number. The random number will be <= b.
Returns
A pseudo-random integer between a and b, between 0 and a, or a float between 0.0 and 1.1 depending on usage.
Note
All parameters are optional.
See also
setRandomSeed

Set the current seed for the random number generator. Based on this seed, a repeatable sequence of numbers will be produced by getRandom().
