Detailed Description

Compute the intersection of the ray that emanates at start in the direction.

Parameters
startPoint where the ray emanates.
dirDirection in which the ray travels. Must be normalized.
outHitResulting intersection point. Only set if there indeed is a hit.
Returns
True if the ray intersects the plane or false if not.
