virtual bool AnimationComponent::playThread ( (int slot=-1, string name="", bool transition=true, float transitionTime=0.5)  )
inlinevirtual

Start a new animation thread, or restart one that has been paused or stopped.

Parameters
slotthread slot to play. Valid range is 0 - 3)
namename of the animation sequence to play in this slot. If not specified, the paused or stopped thread in this slot will be resumed.
Returns
true if successful, false if failed
Example:
%obj.playThread( 0, "ambient" ); // Play the ambient sequence in slot 0
%obj.setThreadTimeScale( 0, 0.5 ); // Play at half-speed
%obj.pauseThread( 0 ); // Pause the sequence
%obj.playThread( 0 ); // Resume playback
%obj.playThread( 0, "spin" ); // Replace the sequence in slot 0
