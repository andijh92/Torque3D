Detailed Description

Implements the duDebugDraw interface in Torque.

This class uses the primitive builder (gfx/primBuild.h) to render navmeshes and other Recast data. To facilitate the primbuilder's requirement to know the number of vertices to render beforehand, this class stores all vertices in a buffer of its own, then passes on that known-size buffer. This means that you only need to call the duDebugDraw functions when your data changes. At other times, you can cache the duDebugDrawTorque object and call its render() method, which actually renders its buffered data.
