An abstract way to raycast into any type of PhysicsWorld, in a way that mirrors a Torque-style raycast. This method is not fully developed or very sophisticated. For example, there is no system of collision groups or raycast masks, which could be very complex to write in a PhysicsPlugin-Abstract way...
