Member Enumeration Documentation

enum BoxBase::Planes

Indices for the side planes of the box. Each pair of planes has successive indices. Also, the planes are ordered by X (left&right), Y (near&far), and Z (top&bottom).

enum BoxBase::Points

Indices of the corner points.

Note
The order defined here is expected by several places in the code!
