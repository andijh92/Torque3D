Detailed Description

A generator that will dump all export structures contained in an engine DLL to an XML file which may then be used by wrapper generators to create a language-specific binding for the engine API. Using XML as an intermediary format allows the generators to use all of the export structures without actually having to access them directly in the DLL as native entities.
