Detailed Description

template<class T>
class ThreadSafeFreeList< T >

Freelist for re-using allocations in a concurrent setting.

Note
Make sure that there are no more allocations in use when the free list is destructed.
Allocated instances come with a reference already counted on the instance.
Parameters
TType of elements to allocate; must be derived from ThreadSafeRefCount and have at least define one additional pointer-sized field.
