Sorting

Before testing, culling volumes will be sorted by decreasing probability of causing test positives. Thus, the sort point of a volume should be a rough metric of the amount of scene/screen space it covers.

Note that sort points for occluders are independent of sort points for includers.

Detailed Description

A volume used to include or exclude space in a scene.

Culling volumes are represented as sets of clipping planes.

Note
Culling is performed in world space so the plane data for culling volumes must be in world space too.
