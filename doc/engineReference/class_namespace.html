Detailed Description

A dictionary of function entries.

Namespaces are used for dispatching calls in the console system.

inline

Decrease the count on the reference that this namespace holds to its parent.

Note

Namespace* Namespace::getPackageRoot ( )
inline

Return the topmost package in the parent hierarchy of this namespace that still refers to the same namespace. If packages are active and adding to this namespace, then they will be linked in-between the namespace they are adding to and its real parent namespace.
