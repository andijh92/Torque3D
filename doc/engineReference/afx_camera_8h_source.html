27 // afxCamera implements a modified camera for demonstrating a third person camera style
28 // which is more common to RPG games than the standard FPS style camera. For the most part,
29 // it is a hybrid of the standard TGE camera and the third person mode of the Advanced Camera
30 // resource, authored by Thomas "Man of Ice" Lund. This camera implements the bare minimum
31 // required for demonstrating an RPG style camera and leaves tons of room for improvement.
32 // It should be replaced with a better camera if possible.
33 //
34 // Advanced Camera Resource by Thomas "Man of Ice" Lund:
35 // http://www.garagegames.com/index.php?sec=mg&mod=resource&page=view&qid=5471
