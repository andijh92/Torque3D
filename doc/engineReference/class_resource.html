template<class T>
static Signal<bool(const Torque::Path &, void**)>& Resource< T >::getLoadSignal ( )
inlinestatic

Registering with this signal will give an opportunity to handle resource creation before calling the create() function. This may be used to handle file extensions differently and allow a more plugin-like approach to adding resources. Using this mechanism, one could, for example, override the default methods for loading DTS without touching the main source.
