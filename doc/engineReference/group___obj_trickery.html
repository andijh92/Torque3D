Detailed Description

These functions are to construct and destruct objects in memory without causing a free or malloc call to occur. This is so that we don't have to worry about allocating, say, space for a hundred NetAddresses with a single malloc call, calling delete on a single NetAdress, and having it try to free memory out from under us.
