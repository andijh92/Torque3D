template<typename T>
U32 ThreadSafeDeque< T >::Node::mIsClaimed

Thread claim flag. This is to prevent two threads who concurrently do a tryPopFront() and tryPopBack() respectively on a deque with just a single node to both claim and return the same value (which would happen without the flag as otherwise both threads would use two different deletion bits for claiming the node).
