Detailed Description

template<class T, class DeletePolicy = DeleteSingle>
class ThreadSafeRefCount< T, DeletePolicy >

Baseclass for concurrently reference-counted objects.

Note
NOTE that freshly instantiated objects start out with a reference count of ZERO! Depending on how this class is used, this may not be desirable, so override this behavior in constructors if necessary.
Parameters
Tthe class being reference counted; this is passed to this class, so it can call the correct destructor without having to force users to have virtual methods
