Detailed Description

Abstract base class for back-end sound API implementations.

Member Enumeration Documentation

enum SFXDevice::ECaps

Device capability flags.

Enumerator
CAPS_Reverb 

Device supports reverb environments.

CAPS_VoiceManagement 

Device manages voices on its own; deactivates virtualization code in SFX system.

CAPS_Occlusion 

Device has its own sound occlusion handling (SFXOcclusionManager).

CAPS_DSPEffects 

Device implements DSP effects (SFXDSPManager).

CAPS_MultiListener 

Device supports multiple listeners.

CAPS_FMODDesigner 

FMOD Designer support.

void SFXDevice::_releaseAllResources ( )
protected

Release all resources tied to the device. Can be called repeatedly without harm. It is meant for device destructors that will severe the connection to the sound API and thus need all resources freed before the base destructor is called.
