Detailed Description

Interface for tracking GFX adapters and initializing them into devices.

Note
Implement this class per platform.
This is just a class so it can be friends with GFXDevice)

GFXAdapter * GFXInit::chooseAdapter ( GFXAdapterType  type,
const char *  outputDevice 
)
static

Chooses a suitable GFXAdapter, based on type, preferences, and fallbacks. If the requested type is omitted, we use the prefs value. If the requested type isn't found, we use fallbacks: OpenGL, NullDevice This method never returns NULL.
