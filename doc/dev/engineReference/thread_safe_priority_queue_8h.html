Detailed Description

Template code for an efficient thread-safe priority queue implementation. There are two alternative implementations to choose from: ThreadSafePriorityQueue and ThreadSafePriorityQueueWithUpdate where the latter adds concurrent status updates of queue items to the former implementation.
