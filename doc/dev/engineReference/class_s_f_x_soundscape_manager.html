Detailed Description

The soundscape manager produces a dynamic mix between multiple active soundscapes.

Note
The more layered soundscapes there are, the costlier ambient sound updates will get.

Stack of soundscape that are currently being faded out. These are kept around so that when their ambient spaces are activated again, they can be brought back with their fades simply being reversed.
