153 // This class contains code for rendering and manipulating a 3D gizmo, it
154 // is usually used as a helper within a TSEdit-derived control.
155 //
156 // The Gizmo has a MatrixF transform and Point3F scale on which it will
157 // operate by passing it Gui3DMouseEvent(s).
158 //
159 // The idea is to set the Gizmo transform/scale to that of another 3D object
160 // which is being manipulated, pass mouse events into the Gizmo, read the
161 // new transform/scale out, and set it to onto the object.
162 // And of course the Gizmo can be rendered.
163 //
164 // Gizmo derives from SimObject only because this allows its properties
165 // to be initialized directly from script via fields.
166
