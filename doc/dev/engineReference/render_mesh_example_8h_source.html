40 // This class implements a basic SceneObject that can exist in the world at a
41 // 3D position and render itself. There are several valid ways to render an
42 // object in Torque. This class implements the preferred rendering method which
43 // is to submit a MeshRenderInst along with a Material, vertex buffer,
44 // primitive buffer, and transform and allow the RenderMeshMgr handle the
45 // actual setup and rendering for you.
46 //-----------------------------------------------------------------------------
