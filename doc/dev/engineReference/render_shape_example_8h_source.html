34 // This class implements a basic SceneObject that can exist in the world at a
35 // 3D position and render itself. There are several valid ways to render an
36 // object in Torque. This class makes use of the "TS" (three space) shape
37 // system. TS manages loading the various mesh formats supported by Torque as
38 // well was rendering those meshes (including LOD and animation...though this
39 // example doesn't include any animation over time).
40 //-----------------------------------------------------------------------------
