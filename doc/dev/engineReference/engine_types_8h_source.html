278 // Temporary helper. Used to strip the return or argument types supplied in a function type
279 // down to the value type and thus account for the discrepancy between (argument/return) value
280 // types and engine types. Don't go for the base type as VC will not allow us to use abstract
281 // types even in return or argument positions of abstract function types (GCC allows it; not
282 // sure what the stance of the ANSI C++ standard towards this is). Also, while it may be tempting
283 // to go for the real return and argument value types here, this would lead to errors as these
284 // are not guaranteed to be any meaningful value or base types to the engine type system.
285 #define T( x ) typename EngineTypeTraits< x >::ValueType
