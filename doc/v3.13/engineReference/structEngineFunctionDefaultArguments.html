Detailed Description

Structure storing the default argument values for a function invocation frame.

Member Function Documentation

const U8* EngineFunctionDefaultArguments::getArgs ( ) const
inline

Return a pointer to the variable-sized array of default argument values.

The arguments must be stored IMMEDIATELY after mNumDefaultArgs. This is a FULL frame and not just the default arguments, i.e. it starts with the first argument that the function takes and ends with the last argument it takes. If the compiler's #pragma pack is buggy, the elements in this structure are allowed to be 4-byte aligned rather than byte-aligned as they should be.

References mNumDefaultArgs.

Member Data Documentation

U32 EngineFunctionDefaultArguments::mNumDefaultArgs

Number of default arguments for the function call frame.

This is NOT the size of the memory block returned by getArgs() and also not the number of elements it contains.
