Detailed Description

template<typename T, class TimeSource = GenericTimeSource< SimMSTimer >>
class InterpolatedChangeProperty< T, TimeSource >

A property that smoothly transitions to new values instead of assuming them right away.

Parameters
TValue type. Must have "interpolate( from, to, factor )" method.
TimeSourceTime source to which interpolation is synchronized.
