Detailed Description

Singleton class that exposes ConsoleStaticFunctions for debug visualizing PostEffects.

Example:
// Script interface...
PfxVis::open( PostEffect ) // Multiple PostEffects can be visualized at the same time
PfxVis::clear() // Clear all visualizer windows
PfxVis::hide() // Hide all windows (are not destroyed)
PfxVis::show()
