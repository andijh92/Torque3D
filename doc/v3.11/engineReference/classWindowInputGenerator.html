bool WindowInputGenerator::lastKeyWasGlobalActionMap ( ) const
inline

Tells us if the last key was used within the global action map.

Returns
true if the key was a global action map key, false otherwise.
Note
Useful and currently used to tell if we just opened the console by using the console key. Currently this is used to fix a bug in SDL but it is not limited to that use.
