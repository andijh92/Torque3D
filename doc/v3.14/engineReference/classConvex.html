)
virtual

This is used by the GJK collision in Vehicle.

The Convex class should add verts, edges, and faces to the passed ConvexFeature that face towards the passed normal vector. Verts added in this way should also be transformed by the passed matrix.

Parameters
matTransform which should be applied to verts added to the ConvexFeature
nNormal vector
cfConvexFeature to add data to.
